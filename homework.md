## 3. What three problems does Git & GitHub solve?

- version control
- work on one project as a team
- remote access

## 4. What happens when you `fork` a repository?

We create our copy of the repository, which will be totally separated from the previous one

## 5. What happens when you clone a repository?

It downloads online files to your local computer and makes a local version

## 6. What is a Pull Request?

A Pull Request is a type of request used to propose changes to the code in a branch. It allows developers to review the changes before merging them into the main branch.
